Tags de Agrupamento

Caixa 01 Caixa 02

Tags de agrupamento são usadas para agrupar conteúdo em um documento. Elas não afetam o conteúdo ou o layout da página, mas podem ser úteis ao estilizar a página com CSS.

Tags de agrupamento incluem:

  • <div> - Define uma divisão ou uma seção em um documento HTML
  • <span> - Define uma seção em um documento
  • <header> - Define um cabeçalho para um documento ou uma seção
  • <main> - Define o conteúdo principal de um documento
  • <footer> - Define um rodapé para um documento ou uma seção
  • <nav> - Define um conjunto de links de navegação
  • <article> - Define um artigo independente e auto-contido
  • <section> - Define uma seção em um documento
  • <aside> - Define conteúdo ao lado do conteúdo da página
  • <details> - Define detalhes adicionais que o usuário pode visualizar ou ocultar
  • <summary> - Define um cabeçalho para o elemento <details>
